Assignments

Homework 1 – Introduction to REST APIs & FHIR

In this exercise you will learn how to communicate with remote servers using REST APIs and the Python requests library. You will retrieve real healthcare data from a FHIR R4 sandbox server, explore the JSON format, and practice all four HTTP methods: GET, POST, PUT, and DELETE.

  • Send GET requests to retrieve jokes and FHIR Patient records
  • Understand JSON as a resource representation
  • Modify an existing patient record with POST (FHIR update)
  • Create a new patient record with PUT (client-assigned ID)
  • Delete a patient record with DELETE
